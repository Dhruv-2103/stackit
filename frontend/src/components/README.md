# StackIT UI Components

This directory contains reusable UI components for the StackIT application.

## Available Components

### Modal.jsx
A reusable modal dialog component with customizable header, content, and actions.

### Confirm.jsx
A specialized modal for confirmation dialogs with predefined types (info, delete, warning, ban).

### Alert.jsx
A component for displaying important messages that require attention.

### Toast.jsx
A component for displaying non-intrusive notifications.

### ToastProvider.jsx
A context provider for managing toast notifications throughout the application.

## Usage

For detailed usage instructions and examples, see the [UI Components Documentation](../docs/UI-Components.md).

## Styling

All components follow the StackIT design system with dark mode styling and consistent colors:

- Background colors: `#0C0C0C` (page bg), `#1C1C1E` (card bg), `#2C2C2E` (input bg)
- Text colors: `#FFFFFF` (white text), `#8E8E93` (muted text)
- Accent colors: `#007AFF` (blue), `#34C759` (green), `#FF3B30` (red), `#FF9F0A` (yellow/orange)

## Animation

Components use consistent animations for a smooth user experience:
- Fade in/out
- Slide transitions
- Loading indicators
